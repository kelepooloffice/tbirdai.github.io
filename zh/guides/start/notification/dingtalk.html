# 配置钉钉

由于钉钉限制移动端无法添加自定义机器人，您需通过PC端完成添加。

# 1. 点击对话窗口加号-【发起群聊】

# 2. 选择创建 【会议群】或【培训群】

# 3. 点击创建，无需添加成员

# 4. 在群窗口点击【设置】图标

# 5. 在【群设置】中点击【智能群助手】

# 6. 点击【添加机器人】

# 7. 点击机器人【设置】按钮

# 8. 选择【自定义】机器人

# 9. 点击【添加】

# 10. 输入一个机器人的名字，安全设置选择自定义关键词，输入 TBird

# 11. 复制 [Webhook] 地址，添加到配置中，点击保存即可添加成功
